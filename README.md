# Message Board
Message board where you can post messages and view other messages.

## Technologies

- Angular
- Bootstrap
- Node
- Express
- Mongo


## Requirements 
- Front end has 2 inputs and Submit Button 
- When message is submitted it is saved in mongo database
- All messages are displayed on the Dom
